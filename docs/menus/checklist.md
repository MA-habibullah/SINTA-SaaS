# Checklist Dokumentasi SINTA-SaaS

Berikut adalah daftar dokumentasi per menu yang telah dibuat:

- [x] `01-Dashboard.md`: Penjelasan halaman beranda utama (Super Admin, Admin, Guru, dll).
- [x] `02-Manajemen-Pengguna.md`: Kelola siswa, karyawan, guru, dan role.
- [x] `03-Master-Data-dan-Kelembagaan.md`: Kelola tahun ajaran, kelas, data sekolah.
- [x] `04-Buku-Induk.md`: Rincian data induk siswa.
- [x] `05-Ekstrakurikuler.md`: Manajemen anggota dan penilaian kegiatan ekstrakurikuler.
- [x] `06-Informasi-dan-Agenda.md`: Manajemen pengumuman dan timeline kalender sekolah.
- [x] `07-Bimbingan-Konseling.md`: Modul pelanggaran, kasus, absensi, prestasi, dan penjurusan PDSS.
- [x] `08-Rapot.md`: Manajemen penilaian dan cetak rapot.
- [x] `09-Tracer-Study.md`: Penelusuran alumni (Kuliah, Bekerja, dll).
- [x] `10-Konfigurasi-Utilitas.md`: Log aktivitas, antrean, sesi pengguna, akses.
- [x] `11-Super-Admin.md`: Monitor server, error log, dan manajemen *tenant* lintas sekolah.
- [x] `12-Autentikasi-dan-Login.md`: Modul login, verifikasi password (Bcrypt), proteksi Session.
- [x] `13-Manajemen-Kurikulum.md`: Alokasi Mapel, penugasan guru pengajar, duplikasi kurikulum antar semester.
- [x] `14-Manajemen-Data-Siswa.md`: Dapur pengolahan formulir siswa (5+ tabel relasional), sistem auto-save draft, cascading dropdown geolokasi, dan kompresi pasfoto.
- [x] `15-Fitur-Utilitas-Massal.md`: Dokumentasi penanganan operasi berskala besar, seperti impor *Excel* dan pengolahan `.zip` untuk *Bulk Upload Photo* profil siswa.
